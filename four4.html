1. html This tag defines the root of an HTML document,
  2. head This tag contains metadata information about the document such as the title and links to stylesheets and scripts
  3. Title This tag defines the title of the document that appears in the browser tab.
  4. body This tag contains the content that is displayed in the browser window, such as text, images, and other HTML elements.
  5. h1 to h6 These tags define headings of different levels, with h1 being the largest and most important.
  6. < p > This tag defines a paragraph of text.
  7. < a >This tag creates a hyperlink to another webpage or resource.
  8. img This tag inserts an image into the document.
  9. < ul > This tag creates an unordered list of items.
  10. < ol > This tag creates an ordered list of items.
  11. li This tag defines an item in a list.
  12. div This tag defines a container for other HTML elements and is often used for styling purposes.
  13. span This tag defines a small portion of text or other HTML elements for styling purposes.
  14. form This tag defines a form for user input, such as text boxes and buttons.
  15. input This tag creates an input field within a form, such as a text box or radio button.
