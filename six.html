HTML
HyperText Markup Language is used to structure and format content on the web.
CSS
Cascading Style Sheets is used for styling and layout of web pages.
JavaScript
JavaScript is a scripting language that is used to create dynamic web content and interactivity.
Node.js
Node.js is a server-side JavaScript runtime environment used for building scalable server applications.
Express
Express is a web application framework for Node.js that facilitates the development of robust APIs and web applications.
React
React is a JavaScript library for building user interfaces in web applications.
Angular
Angular is a JavaScript framework for building dynamic single-page applications (SPAs).
Vue.js
Vue.js is a progressive JavaScript framework for building user interfaces and single-page applications.
MongoDB
MongoDB is a NoSQL database used to store and manage data for web applications.
MySQL
MySQL is a popular open-source relational database management system used for storing and retrieving data.
PostgreSQL
PostgreSQL is an open-source relational database management system known for its reliability and robustness.
Redis
Redis is an in-memory data structure store used as a database, cache, and message broker.
